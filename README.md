Add this line to your <ins>**manifest.json**</ins> file under Packages folder
   ```
   "com.unity.nuget.newtonsoft-json": "3.0.2"
   ```
  So it will be like this  
```
   {
      "dependencies": {
         ...
         "com.unity.nuget.newtonsoft-json": "3.0.2"
      }
   }
```

Then install this repository as "submodule"
