1. Add this line to your manifest.json file under Packages folder
   ```
   "com.unity.nuget.newtonsoft-json": "3.0.2"
   ```

  So it will be like this:
  ```
    {
      "dependencies": {
         ...
          "com.unity.nuget.newtonsoft-json": "3.0.2"
      }
    }
  ```


2. Install this repository as "submodule"
